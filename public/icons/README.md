# PWA Icons

Place your PWA icons here:

- `192.png` - 192x192 pixels
- `512.png` - 512x512 pixels

You can generate these icons using:
- [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- Or any image editor

The icons should feature the "Together with AI" branding with a purple/violet theme color (#8b5cf6).
